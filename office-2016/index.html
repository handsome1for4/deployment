Instruction

1. Extract file
2. Delete officedeploymenttool and configuration.xml you have extracted
3. Create configuration.xml which you have generated, setup.cmd and folder "office" same directory with "setup.exe"
4. Create folder "data" in folder "office"
5. Create folder "build" in folder "data" and move file you downloaded
6. Move 9 files you downloaded to folder "build"
7. Run file "setup.cmd" and wait setup complete
